# Gorilla

> Generate vanilla plugin faster

## How to install and use the generator?

Install the slush:

```sh
$ [sudo] npm install -g slush
```

Install the tangerine generator:s

```sh
$ [sudo] npm install -g slush-gorilla
```

Use the generator:

```sh
$ slush gorilla
```
